# 🚀 DigiTools

## 🌟 About the Project
DigiTools is a premium digital tools marketplace designed for creators, designers, and professionals in Bangladesh.  
It provides access to top-rated software subscriptions, helping users boost productivity and unlock powerful digital tools in one place.

---

## 🛠️ Technology Used
- ⚛️ React.js  
- 🎨 Tailwind CSS + DaisyUI  
- 💡 JavaScript (ES6+)  
- 🔔 React-Toastify (NPM Package)  
- 📦 JSON (for product data)

---

## ✨ Key Features

### 📂 1. All Premium Digital Tools Collection
Get access to top tools like Canva Pro, ChatGPT Plus, and Adobe Photoshop in one platform.

---

### 🛒 2. Smart Cart & Management System
Easily add, remove, and manage your selected tools with real-time cart updates and total pricing.

---

### ⚡ 3. Fast & Easy Subscription System
Smooth and user-friendly checkout system to subscribe to digital tools instantly.

---

## 🚀 Project Highlights
- Modern UI with clean design  
- Responsive layout for all devices  
- Toast notifications for better UX  
- Optimized cart management system
